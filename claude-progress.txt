# Claude Progress Log - kaiyanTool AI内容创作平台

## 项目概述
kaiyanTool 是一个AI驱动的视频内容创作平台，支持剧本创作、分镜生成、AI图像/视频生成等功能。

### 项目技术栈
- 后端: Express + Prisma + TypeScript (端口: 3001)
- 前端: React 19 + Vite + Tailwind (端口: 3000)
- 数据库: PostgreSQL (通过 Prisma)
- AI Providers: OpenAI, Zhipu, Google, Antsk 等

### Agent 工作规范
1. 每次会话开始时读取 feature_list.json
2. 选择一个未完成的功能进行实现
3. 完成后运行测试验证功能正常
4. 更新 feature_list.json 中该功能的 passes 状态为 true
5. 提交 git commit 并更新本文件

### 启动命令
```bash
# 开发服务器
cd apps/api && npm run dev
cd apps/web && npm run dev

# Agent 模式
./agent-start.sh           # 使用默认 Coding 模式
./agent-start.sh --init    # 使用 Initializer 模式初始化
```
