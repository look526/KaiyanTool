# Change: 增强 Analytics 页面用户体验

## Why
当前 Analytics 页面存在以下用户体验问题：
1. 缺乏时间范围选择器，用户无法按需查看不同时段数据
2. 数据卡片不可交互，无法进行数据下钻
3. 缺乏数据刷新功能，无法获取最新数据
4. 趋势图表过于简单，仅使用条形图，可视化效果差
5. 缺乏环比/同比数据对比，难以发现增长趋势
6. 缺乏数据导出功能，不便于汇报和存档

这导致：
- 用户无法灵活分析不同时间段的数据
- 数据可操作性差，无法深入分析
- 数据时效性无法保证
- 数据可视化效果不佳，影响决策

## What Changes

### Phase 1: 核心功能增强
- 添加时间范围选择器（今日/本周/本月/自定义）
- 添加数据刷新按钮（手动刷新 + 自动刷新间隔设置）
- 统计卡片添加环比变化指示（↑/↓ 百分比）
- 统计卡片添加迷你趋势图

### Phase 2: 数据可视化增强
- 使用 Recharts 图表库替代简单条形图
- 添加交互式折线图/面积图
- 添加图例和 Tooltip 交互提示
- 支持多指标对比显示

### Phase 3: 数据操作增强
- 添加数据导出功能（CSV 格式）
- 添加异常告警提示（模型失败率过高时显示警告）
- 添加数据洞察模块（AI 生成数据解读）

## Impact
- 影响范围：
  - 前端：`apps/web/src/pages/AnalyticsPage.tsx`
  - 前端：`apps/web/src/components/analytics/` 目录下组件
  - 后端：`apps/api/src/routes/analytics.ts`（可能需要扩展 API）
- 风险等级：低（纯前端增强，不涉及数据结构变更）
- 兼容性：完全向后兼容
