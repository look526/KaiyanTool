# Spec Delta: Content Management

## ADDED Requirements

### Requirement: Script Input and Editing

The system MUST provide script input and editing functionality with rich text editing and auto-save.

#### Scenario: Create new script

**Given** User is logged in
**And** User has project edit permission
**When** User accesses content edit page
**And** Selects "script" type
**Then** System displays script editor
**And** Provides script template

#### Scenario: Save script

**Given** User is logged in
**And** User has completed script editing
**When** User clicks save button
**Then** System saves script to server
**And** Saves original text
**And** Updates save time

### Requirement: Script Parsing

The system MUST be able to parse script text and extract scenes, characters, dialogue, and action information.

#### Scenario: Parse standard script format

**Given** User is logged in
**And** Script content is in standard format
**When** User clicks parse button
**Then** System extracts all scenes
**And** System extracts all characters
**And** System extracts dialogue and action
**And** Generates structured data

### Requirement: Novel Input and Editing

The system MUST provide novel input and editing functionality with chapter management.

#### Scenario: Create new novel

**Given** User is logged in
**And** User has project edit permission
**When** User accesses content edit page
**And** Selects "novel" type
**Then** System displays novel editor
**And** Provides novel template

#### Scenario: Add chapter

**Given** User is logged in
**And** User is editing novel
**When** User clicks add chapter button
**Then** System creates new chapter
**And** Auto-generates chapter number

### Requirement: Storyline Generation

The system MUST be able to generate storyline based on novel content.

#### Scenario: Generate storyline

**Given** User is logged in
**And** Novel content is saved
**When** User clicks generate storyline button
**Then** System calls AI Agent to analyze novel
**And** Extracts main conflict and theme
**And** Identifies key plots
**And** Generates storyline text

### Requirement: Outline Generation

The system MUST be able to generate detailed outline based on storyline.

#### Scenario: Generate outline

**Given** User is logged in
**And** Storyline is generated
**When** User sets episode count (e.g., 10)
**And** Clicks generate outline button
**Then** System calls AI Agent to generate outline
**And** Divides storyline into specified episodes
**And** Each episode contains detailed scenes
**And** Maintains character consistency

### Requirement: Script Generation (from Outline)

The system MUST be able to generate detailed script from outline.

#### Scenario: Generate script from outline

**Given** User is logged in
**And** Outline is generated
**When** User clicks generate script button
**Then** System calls Director Agent
**And** Converts outline to script format
**And** Adds shot information
**And** Adds character actions and expressions
**And** Generates complete script
