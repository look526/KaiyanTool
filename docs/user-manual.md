# 用户手册

## 1. 系统概述

KaiyanTool 是一个基于 AI 的内容创作平台，专为剧本和小说创作设计。系统集成了先进的 AI 技术，帮助用户快速生成创意内容、优化剧本结构、管理项目资源。

### 1.1 核心功能

- **剧本创作与编辑**：专业的剧本编辑器，支持格式化编辑和实时预览
- **小说解析与管理**：自动解析小说文本，识别章节结构
- **AI 辅助创作**：AI 续写、改写、生成剧本
- **导演 Agent**：基于故事大纲自动生成完整剧本
- **角色管理**：创建和管理角色信息，支持参考图片
- **场景管理**：创建和管理场景信息，支持氛围描述
- **项目管理**：组织和管理多个创作项目
- **团队协作**：支持多用户协作编辑
- **AI 提供商集成**：支持 OpenAI、Google、智普 AI 等多个 AI 提供商

### 1.2 系统架构

- **前端**：React + TypeScript + Tailwind CSS
- **后端**：Express + TypeScript + Prisma ORM
- **数据库**：PostgreSQL
- **AI 集成**：支持多个 AI 提供商的 API

## 2. 用户账户管理

### 2.1 注册账号

1. 访问系统登录页面：http://localhost:3000/login
2. 点击「注册」按钮
3. 填写注册表单：
   - 姓名：您的真实姓名
   - 邮箱：用于登录的邮箱地址
   - 密码：至少 6 位，建议包含字母和数字
4. 点击「创建账户」按钮
5. 系统将自动登录并跳转到项目列表页面

### 2.2 登录系统

1. 访问系统登录页面：http://localhost:3000/login
2. 填写登录表单：
   - 邮箱：您的注册邮箱
   - 密码：您的登录密码
   - 记住我：勾选后下次访问会自动登录
3. 点击「登录」按钮
4. 系统验证成功后跳转到项目列表页面

### 2.3 忘记密码

1. 在登录页面，点击「忘记密码」链接
2. 输入您的注册邮箱
3. 点击「发送重置链接」按钮
4. 查看邮箱，点击重置密码链接
5. 输入新密码并确认
6. 点击「重置密码」按钮
7. 使用新密码登录系统

### 2.4 修改个人信息

1. 点击右上角的用户头像
2. 选择「个人设置」
3. 修改个人信息：
   - 姓名
   - 邮箱
   - 密码
4. 点击「保存」按钮

## 3. 项目管理

### 3.1 创建新项目

1. 在项目列表页面，点击「创建项目」按钮
2. 填写项目信息：
   - 项目名称：项目的名称
   - 项目描述：项目的简要描述
   - 项目类型：选择「剧本」、「小说」或「混合」
3. 点击「创建」按钮
4. 系统将跳转到项目详情页面

### 3.2 查看项目列表

1. 点击左侧导航栏中的「项目」
2. 在项目列表页面，您可以：
   - 查看所有项目的基本信息
   - 搜索项目：使用搜索框
   - 筛选项目：按类型、状态等筛选
   - 排序项目：按创建时间、名称等排序

### 3.3 查看项目详情

1. 在项目列表页面，点击项目卡片
2. 在项目详情页面，您可以：
   - 查看项目的详细信息
   - 管理项目成员
   - 访问项目的各个功能模块

### 3.4 编辑项目信息

1. 在项目详情页面，点击「编辑」按钮
2. 修改项目信息：
   - 项目名称
   - 项目描述
   - 项目类型
   - 项目状态
3. 点击「保存」按钮

### 3.5 删除项目

1. 在项目详情页面，点击「删除」按钮
2. 确认删除操作
3. 项目将被永久删除，无法恢复

## 4. 剧本创作

### 4.1 剧本编辑器

1. 在项目详情页面，点击左侧导航栏中的「剧本」
2. 在剧本编辑器中，您可以：
   - 输入剧本标题
   - 在 Monaco 编辑器中输入剧本内容
   - 使用模板：点击「模板」按钮选择预设模板
   - 导入剧本：点击「导入」按钮上传剧本文件
   - 导出剧本：点击「导出」按钮下载剧本文件

### 4.2 剧本格式

系统支持以下剧本格式：

```
场景1 - 室内，白天

主角A：你好！
主角B：你好！
（两人握手）

场景2 - 室外，夜晚

主角A：今天天气真好！
（夜景描述）
```

### 4.3 AI 辅助功能

#### 4.3.1 AI 续写

1. 在剧本编辑器中，输入一些初始内容
2. 点击「AI续写」按钮
3. 系统将使用 AI 生成后续内容
4. 生成的内容将自动添加到编辑器中

#### 4.3.2 AI 改写

1. 在剧本编辑器中，选择要改写的内容
2. 点击「AI改写」按钮
3. 系统将使用 AI 改写选中的内容
4. 改写后的内容将替换原内容

#### 4.3.3 剧本解析

1. 在剧本编辑器中，输入完整的剧本内容
2. 点击「解析」按钮
3. 系统将解析剧本，识别场景和角色
4. 在预览面板中查看解析结果

### 4.4 剧本查看器

1. 在项目详情页面，点击「剧本管理」
2. 点击剧本名称进入剧本查看页面
3. 在剧本查看页面，您可以：
   - 查看格式化的剧本内容
   - 查看场景列表
   - 查看角色列表
   - 复制剧本内容
   - 导出剧本文件

## 6. 导演 Agent

### 6.1 生成剧本

1. 在项目详情页面，点击左侧导航栏中的「导演」
2. 填写剧本生成表单：
   - 故事大纲：简要描述故事的主要情节
   - 类型：选择剧本类型（如：剧情、喜剧、科幻等）
   - 角色信息：添加主要角色，包括姓名、描述、性格
   - 场景信息：添加主要场景，包括名称、描述、氛围
3. 点击「生成剧本」按钮
4. 系统将使用 AI 生成完整的剧本
5. 查看生成的剧本并进行编辑

### 6.2 生成分镜

1. 在项目详情页面，点击左侧导航栏中的「导演」
2. 选择「生成分镜」选项
3. 输入剧本内容
4. 选择视觉风格
5. 点击「生成分镜」按钮
6. 系统将生成分镜方案
7. 查看分镜列表并进行编辑

### 6.3 优化镜头提示词

1. 在项目详情页面，点击左侧导航栏中的「镜头」
2. 选择一个镜头
3. 点击「优化提示词」按钮
4. 上传参考图片（可选）
5. 点击「优化」按钮
6. 系统将优化镜头的提示词
7. 查看优化结果并应用

## 12. 故障排除

### 12.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 登录失败 | 邮箱或密码错误 | 检查邮箱和密码是否正确 |
| 注册失败 | 邮箱已被注册 | 使用其他邮箱注册 |
| API 请求失败 | 后端服务未启动 | 启动后端服务 `npm run dev` |
| 数据库连接失败 | 数据库配置错误 | 检查 .env 文件中的数据库配置 |
| AI 生成失败 | API Key 错误或余额不足 | 检查 AI 提供商配置 |
| 文件上传失败 | 文件大小超过限制 | 减小文件大小或联系管理员 |
| 页面加载缓慢 | 网络连接问题 | 检查网络连接或刷新页面 |

### 12.2 日志查看

- **前端日志**：打开浏览器开发者工具 → Console 标签页
- **后端日志**：查看后端服务的终端输出
- **错误监控**：在生产环境中，查看 Sentry 仪表板

## 13. 最佳实践

### 13.1 剧本创作技巧

1. **大纲先行**：先创建故事大纲，再填充细节
2. **角色塑造**：为每个角色创建详细的背景故事
3. **场景设计**：注重场景的氛围和细节描述
4. **对话自然**：对话要符合角色性格，避免生硬
5. **节奏控制**：合理安排情节节奏，避免拖沓
6. **使用 AI 辅助**：利用 AI 生成创意，再进行人工优化

### 13.2 项目管理建议

1. **分类管理**：按类型或状态对项目进行分类
2. **定期备份**：定期导出项目内容作为备份
3. **版本控制**：使用不同的剧本版本记录创作过程
4. **团队协作**：明确分工，避免冲突编辑
5. **资源管理**：集中管理参考图片和其他资源

### 13.3 AI 使用技巧

1. **明确指令**：给 AI 提供清晰、具体的指令
2. **迭代优化**：多次使用 AI 生成，逐步优化结果
3. **人工审核**：AI 生成的内容需要人工审核和修改
4. **混合使用**：结合多个 AI 提供商的优势
5. **参数调整**：根据需要调整 AI 模型的参数
