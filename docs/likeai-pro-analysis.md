# LIKE AI.pro 业务逻辑分析报告

## 一、产品概述

LIKE AI.pro 是一个专业级AI影视创作平台，通过AI技术降低影视创作门槛，让个人和小团队也能制作专业级内容。

## 二、核心业务流程

### 2.1 用户创作路径

```
项目初始化 → 风格配置 → 内容导入 → 资源管理 → 分镜制作 → 团队协作 → 数据分析
```

### 2.2 详细步骤

1. **项目创建**
   - 选择视频比例：16:9（横屏）/ 9:16（竖屏）
   - 填写项目基本信息

2. **风格选择**
   - 从9种预设风格中选择
   - 或使用自定义风格

3. **剧本导入**
   - 上传剧本文件
   - 或使用AI生成剧本

4. **资源管理**
   - 配置角色、物品、场景
   - 建立资源库

5. **分镜制作**
   - 在工作台进行分镜设计
   - AI辅助生成分镜

6. **团队协作**
   - 多人协作编辑
   - 版本控制

7. **数据分析**
   - 查看项目统计数据
   - 优化创作流程

## 三、功能模块详解

### 3.1 项目创建模块

#### 视频比例选择
- **16:9 横屏**
  - 适用场景：电影、纪录片、B站等平台
  - 特点：传统电影画幅，适合叙事内容

- **9:16 竖屏**
  - 适用场景：抖音、快手、小红书等短视频平台
  - 特点：移动端优化，适合竖屏浏览

#### 设计亮点
- 在项目创建初期就明确输出格式，避免后期调整
- 支持不同平台的发布需求

### 3.2 风格选择系统

#### 9种预设风格

| 风格类型 | 适用场景 | 视觉特点 |
|---------|---------|---------|
| 电影质感 | 电影短片、高端广告 | 专业光影、电影级调色、高对比度 |
| 高清实拍 | 商业广告、产品展示 | 写实风格、高清晰度、自然色彩 |
| 暗黑哥特 | 悬疑惊悚、奇幻题材 | 低饱和度、神秘氛围、暗色调 |
| 赛博朋克 | 科幻题材、未来感 | 霓虹色调、科技元素、未来感 |
| 日漫风格 | 二次元内容、动漫短剧 | 日系画风、二次元特征、明亮色彩 |
| 新海诚风 | 治愈系、青春题材 | 唯美光影、细腻情感、清新色调 |
| 国风水墨 | 古风、传统文化 | 水墨笔触、东方美学、留白艺术 |
| 游戏原画 | 游戏宣传、概念设计 | 精致细节、游戏感、高完成度 |
| 自定义 | 个性化需求 | 用户自定义风格参数 |

#### 设计亮点
- 风格分类清晰，覆盖主流创作需求
- 支持自定义，满足个性化创作
- 风格选择前置，确保整体视觉统一

### 3.3 剧本管理模块

#### 功能特性
- **上传剧本**：支持多种格式导入（TXT、DOCX等）
- **AI剧本生成**：根据主题自动生成剧本
- **剧本解析**：自动拆解为分镜脚本
- **剧本编辑**：在线编辑和调整

#### 业务逻辑
1. 用户输入主题或上传剧本
2. AI分析剧本内容
3. 自动识别角色、场景、对话
4. 生成分镜脚本框架
5. 用户可手动调整和优化
