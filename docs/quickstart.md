# 快速开始指南

本指南将帮助您快速上手 KaiyanTool 系统，从注册账号到创建第一个项目，让您在几分钟内体验系统的核心功能。

## 1. 系统要求

### 前端要求
- 现代浏览器：Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- 网络连接：稳定的互联网连接
- 分辨率：推荐 1280x720 及以上

### 后端要求
- Node.js 18.0+
- PostgreSQL 14.0+
- 2GB+ 内存
- 20GB+ 存储空间

## 2. 安装与启动

### 本地开发环境

1. **克隆代码库**
   ```bash
   git clone https://github.com/yourusername/kaiyantool.git
   cd kaiyantool
   ```

2. **安装依赖**
   ```bash
   # 安装根目录依赖
   npm install
   
   # 安装前端依赖
   cd apps/web
   npm install --legacy-peer-deps
   
   # 安装后端依赖
   cd ../api
   npm install
   ```

3. **配置环境变量**
   ```bash
   # 在 api 目录创建 .env 文件
   cp .env.example .env
   # 编辑 .env 文件，配置数据库连接和其他参数
   ```

4. **启动服务**
   ```bash
   # 启动后端服务
   cd apps/api
   npm run dev
   
   # 启动前端服务
   cd ../web
   npm run dev
   ```

5. **访问系统**
   - 前端：http://localhost:3000
   - 后端 API：http://localhost:3001

## 3. 首次使用流程

### 3.1 注册账号

1. 打开 http://localhost:3000/login
2. 点击「注册」按钮
3. 填写注册信息：
   - 姓名
   - 邮箱（用于登录）
   - 密码（至少 6 位）
4. 点击「创建账户」按钮
5. 系统将自动登录并跳转到项目列表页面

### 3.2 创建第一个项目

1. 在项目列表页面，点击「创建项目」按钮
2. 填写项目信息：
   - 项目名称
   - 项目描述
   - 项目类型：选择「剧本」
3. 点击「创建」按钮
4. 系统将跳转到项目详情页面

### 3.3 编辑剧本

1. 在项目详情页面，点击左侧导航栏中的「剧本」
2. 在剧本编辑器中：
   - 输入剧本标题
   - 在编辑器中输入剧本内容
   - 支持的格式：
     ```
     场景1 - 室内，白天
     
     主角A：你好！
     主角B：你好！
     （两人握手）
     ```
3. 点击「保存」按钮保存剧本

### 3.4 使用 AI 辅助功能

1. 在剧本编辑器中，输入一些初始内容
2. 点击「AI续写」按钮，系统将自动续写剧本
3. 点击「AI改写」按钮，系统将自动改写选中的内容
4. 点击「解析」按钮，系统将解析剧本并显示场景和角色

### 3.5 查看剧本

1. 保存剧本后，系统将创建剧本记录
2. 在项目详情页面，点击「剧本管理」查看所有剧本
3. 点击剧本名称进入剧本查看页面
4. 在剧本查看页面，您可以：
   - 查看格式化的剧本内容
   - 查看角色列表
   - 复制或导出剧本

## 4. 核心功能体验

### 4.1 小说解析

1. 在项目详情页面，点击左侧导航栏中的「小说」
2. 点击「创建小说」按钮
3. 在小说编辑器中，粘贴小说内容
4. 点击「解析」按钮，系统将自动识别章节
5. 查看解析后的章节列表

### 4.2 导演 Agent

1. 在项目详情页面，点击左侧导航栏中的「导演」
2. 填写剧本生成信息：
   - 故事大纲
   - 类型
   - 角色信息
   - 场景信息
3. 点击「生成剧本」按钮
4. 系统将使用 AI 生成完整的剧本
5. 查看生成的剧本并进行编辑

### 4.3 角色管理

1. 在项目详情页面，点击左侧导航栏中的「角色」
2. 点击「创建角色」按钮
3. 填写角色信息：
   - 角色名称
   - 角色描述
   - 角色外观
4. 上传角色参考图片
5. 保存角色信息

### 4.4 场景管理

1. 在项目详情页面，点击左侧导航栏中的「场景」
2. 点击「创建场景」按钮
3. 填写场景信息：
   - 场景名称
   - 场景描述
   - 场景时间
   - 场景氛围
4. 上传场景参考图片
5. 保存场景信息

## 5. AI 提供商配置

### 5.1 添加 AI 提供商

1. 点击左侧导航栏中的「设置」→「AI 提供商」
2. 点击「添加提供商」按钮
3. 填写提供商信息：
   - 提供商名称
   - 提供商类型：选择 OpenAI、Google 或 智普 AI
   - API Key
   - 模型列表
4. 点击「测试连接」按钮验证配置
5. 点击「保存」按钮

### 5.2 常用模型推荐

- **OpenAI**: gpt-4o, gpt-3.5-turbo
- **Google**: gemini-1.5-pro
- **智普 AI**: glm-4

## 6. 故障排除

### 6.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 登录失败 | 邮箱或密码错误 | 检查邮箱和密码是否正确 |
| 注册失败 | 邮箱已被注册 | 使用其他邮箱注册 |
| API 请求失败 | 后端服务未启动 | 启动后端服务 `npm run dev` |
| 数据库连接失败 | 数据库配置错误 | 检查 .env 文件中的数据库配置 |
| AI 生成失败 | API Key 错误或余额不足 | 检查 AI 提供商配置 |

### 6.2 日志查看

- **前端日志**: 浏览器开发者工具 → Console
- **后端日志**: 后端终端输出
- **错误监控**: Sentry 仪表板（生产环境）
