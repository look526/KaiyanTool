# 分镜页面资产生成实施计划

## 需求说明

在脚本编辑器页面（`/projects/:projectId/script`）生成资产时，需要将**剧集、场景、分镜**都生成到分镜页面（`/projects/:id/shots`）里面去。

### 当前问题

1. **脚本编辑器已能生成资产**：`ScriptEditorPage.tsx` 中的 `handleGenerateAssets()` 函数已经可以生成角色、场景、分镜
2. **分镜页面独立管理**：分镜页面使用 `EpisodesPage.tsx` 管理剧集，支持剧集 → 场景 → 分镜三层结构
3. **数据未打通**：脚本编辑器生成的分镜没有关联到剧集，导致分镜页面看不到

### 目标

在脚本编辑器页面点击"生成资产"后：
1. 自动创建一个默认剧集（Episode）
2. 生成的场景关联到该剧集
3. 生成的分镜关联到对应场景和剧集
4. 在分镜页面可以看到所有生成的内容

---

## 技术方案

### 方案一：创建默认剧集（推荐）⭐

**优点：**
- 改动最小
- 用户体验简单，一键生成
- 保持数据结构一致性

**实现步骤：**

#### 1. 后端 API 改造

**文件：** `apps/api/src/controllers/shot.controller.ts`

创建新的 API 端点：`POST /api/shots/projects/:projectId/generate-from-script`

```typescript
/**
 * 从剧本批量生成剧集、场景、分镜
 * 请求体：
 * {
 *   scenes: Array<{
 *     heading: string;
 *     description: string;
 *     time: string;
 *     dialogues: Array<{ characterName: string; text: string }>;
 *     actions: Array<{ description: string; shot?: { type: string; movement: string } }>;
 *   }>;
 *   characters: Array<{ name: string; appearance?: object; description?: string }>;
 *   items: Array<{ name: string; size?: string; shape?: string; color?: string }>;
 * }
 */
export const generateFromScript = async (req: Request, res: Response) => {
  const { projectId } = req.params;
  const { scenes, characters, items } = req.body;
  
  // 1. 创建默认剧集
  const episode = await prisma.episode.create({
    data: {
      project_id: projectId,
      title: '第 1 集',
      episode_number: 1,
      description: '由剧本自动生成',
    },
  });
  
  // 2. 批量创建角色
  for (const char of characters) {
    await prisma.character.create({
      data: {
        project_id: projectId,
        name: char.name,
        appearance: char.appearance ? JSON.stringify(char.appearance) : null,
        description: char.description || '',
      },
    });
  }
  
  // 3. 批量创建物品
  for (const item of items) {
    await prisma.item.create({
      data: {
        project_id: projectId,
        name: item.name,
        type: '其他',
        description: `尺寸：${item.size || '未知'}, 形状：${item.shape || '未知'}, 颜色：${item.color || '未知'}`,
      },
    });
  }
  
  // 4. 批量创建场景和分镜
  for (let i = 0; i < scenes.length; i++) {
    const scene = scenes[i];
    
    // 创建场景
    const createdScene = await prisma.scene.create({
      data: {
        project_id: projectId,
        episode_id: episode.id,
        location: scene.heading?.substring(0, 30) || `场景 ${i + 1}`,
        time: scene.time || '白天',
        description: scene.description || scene.heading || '',
        scene_order: i,
      },
    });
    
    // 创建对话分镜
    const dialogues = scene.dialogues || [];
    for (let j = 0; j < dialogues.length; j++) {
      const dialogue = dialogues[j];
      await prisma.shot.create({
        data: {
          project_id: projectId,
          episode_id: episode.id,
          scene_id: createdScene.id,
          chapter_number: i + 1,
          episode_number: 1,
          segment_id: 1,
          cell_id: j + 1,
          action_summary: `${dialogue.characterName}: "${dialogue.text}"`,
          start_prompt: buildPrompt(scene, dialogue.characterName, dialogue.text, 'start'),
          end_prompt: buildPrompt(scene, dialogue.characterName, dialogue.text, 'end'),
          duration: Math.max(3, Math.ceil(dialogue.text.length / 8)),
          aspect_ratio: '16:9',
          camera_movement: '中景，固定，平视',
          status: 'pending',
        },
      });
    }
    
    // 创建动作分镜
    const actions = scene.actions || [];
    for (let k = 0; k < actions.length; k++) {
      const action = actions[k];
      if (action.shot) {
        await prisma.shot.create({
          data: {
            project_id: projectId,
            episode_id: episode.id,
            scene_id: createdScene.id,
            chapter_number: i + 1,
            episode_number: 1,
            segment_id: 1,
            cell_id: dialogues.length + k + 1,
            action_summary: action.description || '动作描述',
            start_prompt: buildPrompt(scene, '', action.description, 'start'),
            end_prompt: buildPrompt(scene, '', action.description, 'end'),
            duration: action.shot?.duration || 3,
            aspect_ratio: '16:9',
            camera_movement: action.shot 
              ? `${action.shot.type || '中景'},${action.shot.movement || '固定'},${action.shot.angle || '平视'}`
              : '中景，固定，平视',
            status: 'pending',
          },
        });
      }
    }
  }
  
  res.json({
    success: true,
    episode_id: episode.id,
    scenes_count: scenes.length,
    characters_count: characters.length,
    items_count: items.length,
  });
};

function buildPrompt(scene, characterName, text, type) {
  const location = scene.description || scene.heading || '室内';
  const time = scene.time || '白天';
  
  if (type === 'start') {
    return `【场景】${location}，${time}\n【角色】${characterName}\n【台词】${text}`;
  } else {
    return `【场景】${location}，${time}\n【角色】${characterName}说完台词`;
  }
}
```

```typescript
export interface GenerateFromScriptInput {
  scenes: Array<{
    heading: string;
    description: string;
    time: string;
    dialogues: Array<{ characterName: string; text: string }>;
    actions: Array<{ description: string; shot?: { type: string; movement: string; angle: string; duration: number } }>;
  }>;
  characters: Array<{ name: string; appearance?: any; description?: string }>;
  items: Array<{ name: string; size?: string; shape?: string; color?: string }>;
}

export interface GenerateFromScriptResult {
  success: boolean;
  episode_id: string;
  scenes_count: number;
  characters_count: number;
  items_count: number;
}

export const shotsApi = {
  // ... 现有方法
  
  /**
   * 从剧本批量生成剧集、场景、分镜
   */
  async generateFromScript(
    projectId: string,
    input: GenerateFromScriptInput
  ): Promise<GenerateFromScriptResult> {
    const response = await apiClient.post(
      `/shots/projects/${projectId}/generate-from-script`,
      input
    );
    return response.data.data || response.data;
  },
};
```

#### 4. 修改脚本编辑器页面

**文件：** `apps/web/src/pages/ScriptEditorPage.tsx`

修改 `handleGenerateAssets` 函数：

```typescript
const handleGenerateAssets = async () => {
  if (!projectIdStr || parsedScenes.length === 0) {
    addToast({ type: 'warning', title: '无法生成', message: '请先解析剧本' });
    return;
  }
  
  const taskId = addTask('converting', '生成资产中...');
  
  try {
    setIsGeneratingAssets(true);
    const stopProgress = simulateProgress(taskId, 5000);
    
    // 调用新的 API
    const result = await apiClient.generateFromScript(projectIdStr, {
      scenes: parsedScenes,
      characters: characters,
      items: parsedItems,
    });
    
    stopProgress();
    updateTask(taskId, { status: 'completed', progress: 100 });
    
    addToast({ 
      type: 'success', 
      title: '生成完成', 
      message: `已创建 1 个剧集、${result.scenes_count} 个场景、${result.characters_count} 个角色、${result.items_count} 个物品` 
    });
    
    // 可选：自动跳转到分镜页面
    // navigate(`/projects/${projectIdStr}/shots`);
    
  } catch (error) {
    updateTask(taskId, { status: 'failed', error: error instanceof Error ? error.message : '未知错误' });
    addToast({ type: 'error', title: '生成失败' });
  } finally {
    setIsGeneratingAssets(false);
  }
};
```

## 数据流程图

```
用户操作
  ↓
点击"生成资产"
  ↓
ScriptEditorPage.handleGenerateAssets()
  ↓
调用 apiClient.generateFromScript()
  ↓
后端 /api/shots/projects/:projectId/generate-from-script
  ↓
1. 创建默认 Episode
2. 批量创建 Characters
3. 批量创建 Items
4. 对每个 Scene：
   - 创建 Scene（关联 episode_id）
   - 对每个 Dialogue：创建 Shot（关联 scene_id + episode_id）
   - 对每个 Action：创建 Shot（关联 scene_id + episode_id）
  ↓
返回 { episode_id, scenes_count, ... }
  ↓
前端显示成功提示
  ↓
用户访问 /projects/:id/shots
  ↓
看到生成的剧集、场景、分镜
```

## 注意事项

1. **事务处理**：批量创建应使用 Prisma 事务，确保数据一致性
   ```typescript
   await prisma.$transaction(async (tx) => {
     // 所有创建操作
   });
   ```

2. **错误处理**：部分失败时的回滚策略

3. **性能优化**：大量分镜时的批量插入
   ```typescript
   await prisma.shot.createMany({ data: [...] });
   ```

4. **用户体验**：
   - 显示生成进度
   - 提供"查看分镜"按钮
   - 支持取消操作

5. **权限验证**：确保用户有项目访问权限

## 总结

采用**方案一（创建默认剧集）**可以快速实现需求，改动最小，风险最低。后续可以根据用户需求扩展到方案二。

核心思路：
- **后端**：新增批量生成 API，自动创建剧集并关联所有数据
- **前端**：调用新 API，保持用户操作不变
- **数据**：通过 `episode_id` 和 `scene_id` 维护层级关系
