# 小说转剧本格式功能 - 详细设计计划书

## 1. 需求概述

在剧本编辑器AI助手面板中，替代现有的"小说转剧本"功能，将小说文本转换成 `剧本.txt` 格式的纯文本。

### 1.1 业务流程

```
用户输入(集数+分钟数)
    ↓
调用后端API
    ↓
AI自动判断划分集数、场次、人物、场景
    ↓
返回格式化后的纯文本
    ↓
右下角显示进度
    ↓
弹窗询问"是否保存为剧本"
    ↓
用户确认 → 替换编辑器内容
```

### 1.2 目标格式示例

```
《末日饭店》
人物小传：
路遥（25岁）
女主，重生者，末日饭店老板...

第一集
1-1 末日饭店 日 外
出场人物：路遥、陆明
【阴森古堡饭店（阴森）门口，路遥扎高马尾...】
路遥：哎？都站住！你们跑了，我这蛋炒饭给谁吃啊！
【切入闪回】
...
```

### 2.2 API 设计

**请求**
```
POST /api/script/format-to-script
Content-Type: application/json

{
  "content": "小说文本内容",
  "episodes": 12,
  "minutesPerEpisode": 45,
  "model": "glm-4"  // 可选
}
```

**响应**
```
{
  "success": true,
  "formattedText": "《标题》\n第一集\n1-1 场景...",
  "metadata": {
    "episodes": 12,
    "totalScenes": 45
  }
}
```

## 7. 关键实现细节

### 7.1 提示词设计要点

```typescript
export const FORMAT_TO_SCRIPT_PROMPTS = {
  systemPrompt: `你是一个专业的剧本格式转换助手。你的任务是将小说文本转换为标准的剧本格式。

剧本格式要求：
1. 集数划分：根据目标集数和每集时长自动划分
2. 场次编号：格式为"集数-序号"（如1-1, 1-2, 2-1）
3. 场景标注：场景名 + 时间（日/夜/晨/昏）+ 内外景（内/外）
4. 出场人物：在每场开头标注
5. 动作描述：用【】包围
6. 对话格式：角色名：对话内容
7. 画外音：用vo表示

请严格按照上述格式输出，不要添加额外说明。`,

  userPromptTemplate: `请将以下小说文本转换为剧本格式：

**目标要求**
- 集数：{{episodes}}集
- 每集时长：{{minutesPerEpisode}}分钟

**小说文本**
{{novelText}}

请直接输出格式化后的剧本文本：`
};
```
