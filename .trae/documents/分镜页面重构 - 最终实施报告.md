### API 端点清单

#### Episode API
```
GET    /api/projects/:projectId/episodes          - 获取剧集列表
POST   /api/projects/:projectId/episodes          - 创建剧集
GET    /api/episodes/:id                          - 获取剧集详情
PUT    /api/episodes/:id                          - 更新剧集
DELETE /api/episodes/:id                          - 删除剧集
GET    /api/episodes/:id/stats                    - 获取剧集统计
```

#### Scene API
```
GET    /api/episodes/:episodeId/scenes            - 获取场景列表
POST   /api/episodes/:episodeId/scenes            - 创建场景
GET    /api/scenes/:id                            - 获取场景详情
PUT    /api/scenes/:id                            - 更新场景
DELETE /api/scenes/:id                            - 删除场景
PUT    /api/scenes/:id/reorder                    - 场景重排序
```

#### ShotAlternative API
```
GET    /api/shots/:shotId/alternatives            - 获取备选视频列表
POST   /api/shots/:shotId/alternatives            - 创建备选视频
PUT    /api/shots/:shotId/alternatives/:id/recommend - 设为推荐
DELETE /api/shots/:shotId/alternatives/:id        - 删除备选视频
```

#### ShotDraft API
```
GET    /api/episodes/:episodeId/drafts            - 获取草稿列表
POST   /api/episodes/:episodeId/drafts            - 保存草稿
PUT    /api/drafts/:id                            - 更新草稿
DELETE /api/drafts/:id                            - 删除草稿
```

#### Mention API
```
GET    /api/projects/:projectId/mentions          - 获取可提及的资源
```
